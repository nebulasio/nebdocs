# 星云技术委员会(NTC)

## 我们的目标

星云技术委员会秉持着开放、共享、透明的精神，致力于推动星云链技术研发逐步去中心化、社区化。区块链技术为建设全新的、自激励的开源社区带来可能性。星云链的价值发现体系、自进化和自激励的技术理念为构建去中心化协作的世界提供了保障。星云技术委员会将全力推动星云愿景的实现。

了解更新星云技术委员会的链接，请[点击](https://nebulas.io/cn/team.html).
